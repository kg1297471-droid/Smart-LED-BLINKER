# Smart LED Blinker using ESP32

## Overview

This project demonstrates a basic Smart Home Automation System using an ESP32 microcontroller. The ESP32 creates a web server that allows users to control connected devices through a web browser over Wi-Fi.

## Features

* Wireless control through Wi-Fi
* Web-based user interface
* Control multiple devices
* Easy to expand with sensors and relays
* Beginner-friendly IoT project

## Components Required

* ESP32 Development Board
* LEDs or Relay Module
* Jumper Wires
* Breadboard
* Wi-Fi Network

## Working Principle

1. ESP32 connects to a Wi-Fi network.
2. A web server is created on the ESP32.
3. Users access the ESP32 IP address through a browser.
4. Button clicks on the webpage send commands to the ESP32.
5. ESP32 switches connected devices ON or OFF.
